> ## NIGHT ACTION PRIORITY / CONFLICT RULES
>
> Execute in this exact order each night:
>
> ```
> 1. Kumarbaz swap (nightOrder: 1) — permanent role swap applied immediately
> 2. Kıskanç Komşu target selection (nightOrder: 2) — stores target, executes at end
> 3. Evil team kill vote (nightOrder: 3) — target selected, not executed yet
> 4. Kapıcı lock (nightOrder: 4) — locked houses recorded
> 5. Otacı Teyze protect (nightOrder: 5) — protections recorded
> 6. Bekçi query (nightOrder: 6) — blocked if target house locked
> 7. Falcı query (nightOrder: 7) — blocked if target house locked, 20% wrong result
> 8. Hoca protect (nightOrder: 8) — overrides Kapıcı lock
> 9. Kahraman Dede kill (nightOrder: 9) — blocked by Otacı/Hoca, NOT by Kapıcı
> 10. Anonim mark (nightOrder: 10)
> 11. Kıskanç Komşu copy executes — copies action of their target
>
> THEN resolve deaths:
> - Evil kill: blocked if target has Otacı/Hoca protection OR Kapıcı lock
> - Kahraman Dede kill: blocked if target has Otacı/Hoca protection
> - Kırık Kalp chain: if aşık died, Kırık Kalp also dies
> - Check win conditions
> - Generate morning announcement
> ```

> ## WIN CONDITION CHECK
>
> Run `checkWinConditions()` after every death (night or lynch):
>
> ```javascript
> // Priority order if multiple trigger simultaneously:
> // 1. Anonim, 2. Kırık Kalp, 3. Kahraman Dede, 4. Evil team, 5. Mahalle
>
> function checkWinConditions(gameState) {
>   const alive = gameState.players.filter(p => p.alive);
>   const evilAlive = alive.filter(p => p.team === 'kotu').length;
>   const goodAlive = alive.filter(p => p.team === 'mahalle').length;
>
>   // Anonim: 3 marked players lynched and Anonim alive
>   if (gameState.anonimMarksLynched >= 3 && isAlive('anonim')) return 'anonim_wins';
>
>   // Kırık Kalp: both Kırık Kalp and aşık alive at game end
>   if (gameState.ended && isAlive('kirik_kalp') && isAlive(gameState.kirikKalpAsik)) return 'kirik_kalp_wins';
>
>   // Kahraman Dede: last player standing
>   if (alive.length === 1 && alive[0].id === 'kahraman_dede') return 'kahraman_dede_wins';
>
>   // Kumarbaz: alive in last 3 players
>   if (alive.length <= 3 && alive.find(p => p.id === 'kumarbaz')) return 'kumarbaz_wins';
>
>   // Evil team wins: evil >= good
>   if (evilAlive >= goodAlive && evilAlive > 0) return 'evil_wins';
>
>   // Sahte Dernek Başkanı lynched: instant evil win (handled separately on lynch)
>
>   // Mahalle wins: all evil and dangerous neutrals eliminated
>   const dangerousNeutrals = alive.filter(p => ['kahraman_dede', 'anonim'].includes(p.id)).length;
>   if (evilAlive === 0 && dangerousNeutrals === 0) return 'mahalle_wins';
>
>   return null; // game continues
> }
> ```
