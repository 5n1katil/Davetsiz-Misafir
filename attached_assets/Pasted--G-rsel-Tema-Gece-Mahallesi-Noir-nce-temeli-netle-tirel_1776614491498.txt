> **ANİMASYONLAR — CSS keyframes ile uygula:**
>
> 1. Faz geçişi (gündüz→gece):
> Body background `#0A0614`'e geçer, 1.5s ease-in-out. Gece ekranında arkaplan SVG: titreşen sokak lambası, rastgele yanıp sönen pencereler (her pencere bağımsız @keyframes, 3-7s aralık, opacity 0.05–0.32 arası).
>
> 2. Eleme duyurusu:
> Kırmızı overlay flash (0.3s), oyuncu kartı `opacity: 0.4` + `filter: grayscale(0.6)` + üstüne `#C8102E` çapraz çizgi (0.8s ease-out). Sonra rol rozeti slide-in (0.4s).
>
> 3. Rol seçim kartları:
> 3 kart translateY(60px) → translateY(0), staggered 0.1s gecikmeyle, 0.5s ease-out. Hafif rotateZ(-3deg, 0deg, 3deg) fan efekti.
>
> 4. Oylama bar'ları:
> width 0 → hedef yüzde, 0.3s ease-out, her yeni oy geldiğinde tetiklenir. Bar fill color `#C8102E`.
>
> 5. Gece aksiyonu onayı:
> Gönder butonunda teal pulse ring: box-shadow 0 0 0 0 `#1ECBE1` → 0 0 0 12px transparent, 0.6s.
>
> 6. Oyun sonu:
> Kazanan rengiyle full-screen radial burst (scale 0→3, opacity 1→0, 1.2s). Arkasında confetti: 20 adet küçük div, random color (fraksiyon renkleri), random trajectory, 2s fall.
>
> 7. Lobby bekleme:
> Hazır olan oyuncular için checkmark pop: scale 0→1.2→1, 0.3s spring.
>
> 8. Timer kritik (son 10 saniye):
> `#F5C842` → `#C8102E` renk geçişi, hafif shake animasyonu 0.5s'de bir.
>
