> **DÜZELTME 1 — Türkçe büyük harf karakterleri**
>
> Tüm `toUpperCase()` çağrılarını Türkçe locale ile değiştir. Uygulama genelinde:
>
> ```typescript
> // Yardımcı fonksiyon — utils.ts'e ekle ve her yerde kullan
> export function toUpperTR(str: string): string {
>   return str.replace(/i/g, 'İ').replace(/ı/g, 'I').toUpperCase();
> }
>
> // Kullanım örnekleri:
> // "mahalleli" → "MAHALLELİ" ✓
> // "oyun ayarları" → "OYUN AYARLARI" ✓
> // "aktif özel roller" → "AKTİF ÖZEL ROLLER" ✓
> ```
>
> LobbyScreen'de tüm section başlıklarında `toUpperTR()` kullan:
> ```tsx
> <Text style={styles.sectionTitle}>{toUpperTR('mahalleli')}</Text>
> <Text style={styles.sectionTitle}>{toUpperTR('oyun ayarları')}</Text>
> <Text style={styles.sectionTitle}>{toUpperTR('aktif özel roller')}</Text>
> // vb.
> ```

> **2a — Paket seçilince alt roller otomatik güncellenir:**
>
> ```typescript
> // Paket içerikleri — LobbyScreen içinde sabit tanım
> const PACKAGE_ROLES = {
>   standard: {
>     mahalle: ['bekci', 'otaci', 'kapici'],
>     cete: [],
>     kaos: [],
>     tarafsiz: []
>   },
>   advanced: {
>     mahalle: ['bekci', 'otaci', 'kapici', 'falci', 'muhtar', 'muhabir', 'hoca'],
>     cete: ['sahte_dernek', 'icten_pazarlikli'],
>     kaos: ['kirik_kalp', 'dedikoducu'],
>     tarafsiz: []
>   },
>   all: {
>     mahalle: ['bekci', 'otaci', 'kapici', 'falci', 'muhtar', 'muhabir', 'hoca', 'tiyatrocu'],
>     cete: ['sahte_dernek', 'icten_pazarlikli'],
>     kaos: ['kumarbaz', 'kiskanc_komsu', 'kirik_kalp', 'dedikoducu'],
>     tarafsiz: ['anonim', 'kahraman_dede']
>   }
> };
>
> // Paket değişince disabledRoles otomatik hesaplanır
> function handlePackageChange(pkg: 'standard' | 'advanced' | 'all') {
>   const allowed = PACKAGE_ROLES[pkg];
>   const allAllowed = [
>     ...allowed.mahalle,
>     ...allowed.cete,
>     ...allowed.kaos,
>     ...allowed.tarafsiz
>   ];
>   // ALL_SPECIAL_ROLES'dan allowed'da olmayanlar disabled
>   const allSpecial = ALL_SPECIAL_ROLES.map(r => r.id);
>   const newDisabled = allSpecial.filter(id => !allAllowed.includes(id));
>
>   updateSettings({
>     rolePackage: pkg,
>     disabledRoles: newDisabled
>   });
> }
> ```

> **DÜZELTME 3 — Rol dağıtım modu seçeneği**
>
> LobbyScreen ayarlarına yeni bir ayar ekle:
>
> ```tsx
> <SettingRow
>   label={toUpperTR('rol dağıtım modu')}
>   tooltip="Rastgele: Herkes oyun başında otomatik rol alır, seçim ekranı açılmaz. Sıralı Seçim: Oyuncular sırayla 3 kart arasından kendi rolünü seçer, kim seçtiği gizli kalır."
>   options={['random', 'pick']}
>   labels={['Rastgele Dağıt', 'Sıralı Seçim']}
>   value={settings.roleDistribution ?? 'pick'}
>   onChange={v => updateSetting('roleDistribution', v)}
> />
> ```
>
> `RoomSettings` interface'e ekle:
> ```typescript
> roleDistribution?: 'random' | 'pick'; // default: 'pick'
> ```
>
> `gameEngine.ts` — `startGame()` içinde:
> ```typescript
> if (room.settings.roleDistribution === 'random') {
>   // Havuzu karıştır ve direkt ata — ROLE_SELECT fazını atla
>   const shuffledPool = [...room.rolePool].sort(() => Math.random() - 0.5);
>   room.players.forEach((p, i) => {
>     p.roleId = shuffledPool[i];
>     p.roleSelectStatus = 'done';
>   });
>   // ROLE_SELECT'i tamamen atla, doğrudan ROLE_REVEAL'e geç
>   enterRoleRevealPhase(room);
> } else {
>   // Mevcut sıralı seçim akışı — değişiklik yok
>   enterRoleSelectPhase(room);
> }
> ```
>
