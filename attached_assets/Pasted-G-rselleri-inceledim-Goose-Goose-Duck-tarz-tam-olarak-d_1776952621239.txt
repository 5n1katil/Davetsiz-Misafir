> In `gameEngine.ts`, `buildCurrentChoices()` veya rol seçim kartlarını hazırlayan fonksiyonda:
>
> ```typescript
> function buildChoicesForPlayer(room: Room, playerId: string): string[] {
>   // Havuzdan henüz seçilmemiş roller
>   const remaining = room.rolePool.filter(r => !room.assignedRoles.includes(r));
>
>   // Tekrar eden rolleri teke indir (ör: 3 koylu varsa 1 koylu göster)
>   const unique = [...new Set(remaining)];
>
>   // Karıştır
>   const shuffled = unique.sort(() => Math.random() - 0.5);
>
>   // Max 3 seçenek — ama unique sayısı 3'ten azsa o kadar göster (min 1)
>   return shuffled.slice(0, Math.min(3, shuffled.length));
> }
> ```
>

> ```tsx
> // 3 seçenek varsa 3 kart, 2 varsa 2 kart, 1 varsa 1 kart + zorunlu seç

> **Oyuncu listesinde isim gizleme — önemli kural:**
>
> Sol panelde oyuncu listesinde:
> - İsimler HİÇ gösterilmez
> - Sadece sıra numarası: OYUNCU 01, OYUNCU 02...
> - Seçim yapıldıktan sonra: rol emoji + rol adı görünür
> - Hangi numaranın kim olduğu bilinmez
> - Tek istisna: kendi sıran — "sen" ibaresi gösterilebilir opsiyonel
>
> ```typescript
> // gameEngine.ts publicView() içinde, ROLE_SELECT fazında:
> // Diğer oyuncuların selectedRoleId'sini göster ama nickname sıralamasını karıştır
>
> // publicView'da roleSelect fazı için özel mapping:
> if (room.phase === 'ROLE_SELECT') {
>   // Oyuncuları karıştırılmış sırayla ver (gerçek sıra gizli)
>   // Sadece: seçim yaptı mı (done/picking/waiting) + seçtiyse hangi rol
>   // nickname'i VERME — sadece shuffledIndex ver
>   return room.players.map((p, i) => ({
>     shuffledIndex: i, // gerçek sıra değil, görsel sıra
>     status: getStatus(p),
>     selectedRoleId: p.status === 'done' ? p.selectedRoleId : null,
>     isMe: p.id === viewerId,
>   }));
> }
> ```
